  GUIDE : Convertir le fichier HTML en Document Word (.docx)
═══════════════════════════════════════════════════════════════════════

📄 Fichier HTML créé : Guide_Export_Inventaire.html

Vous avez 3 options pour obtenir un document Word :

───────────────────────────────────────────────────────────────────────
OPTION 1 : Ouvrir avec Microsoft Word (RECOMMANDÉ)
───────────────────────────────────────────────────────────────────────

1. Clic droit sur "Guide_Export_Inventaire.html"
2. Ouvrir avec → Microsoft Word
3. Fichier → Enregistrer sous
4. Type : Document Word (.docx)
5. Sauvegarder

✅ Avantages :
   - Préserve toute la mise en forme
   - Tableaux bien formatés
   - Couleurs conservées
   - Prêt à modifier

───────────────────────────────────────────────────────────────────────
OPTION 2 : Via LibreOffice Writer (GRATUIT)
───────────────────────────────────────────────────────────────────────

1. Ouvrir LibreOffice Writer
2. Fichier → Ouvrir → Guide_Export_Inventaire.html
3. Fichier → Exporter au format PDF
   OU
   Fichier → Enregistrer sous → Format Microsoft Word (.docx)

✅ Avantages :
   - Solution gratuite
   - Bonne compatibilité
   - Export PDF aussi possible

───────────────────────────────────────────────────────────────────────
OPTION 3 : Via Google Docs (EN LIGNE)
───────────────────────────────────────────────────────────────────────

1. Aller sur https://docs.google.com
2. Fichier → Ouvrir
3. Onglet "Importer"
4. Sélectionner Guide_Export_Inventaire.html
5. Fichier → Télécharger → Microsoft Word (.docx)

✅ Avantages :
   - Accessible de partout
   - Pas d'installation requise
   - Partage facile

───────────────────────────────────────────────────────────────────────
VÉRIFICATION APRÈS CONVERSION
───────────────────────────────────────────────────────────────────────

Après conversion, vérifiez :

☐ Toutes les pages sont présentes
☐ Les tableaux sont bien formatés
☐ Les couleurs sont conservées (titres, écarts)
☐ Les listes à puces sont correctes
☐ Les en-têtes (h1, h2, h3) sont distincts
☐ Sauts de page aux bons endroits

───────────────────────────────────────────────────────────────────────
PERSONNALISATION DU DOCUMENT WORD
───────────────────────────────────────────────────────────────────────

Une fois en .docx, vous pouvez :

✓ Ajouter votre logo entreprise
✓ Modifier les couleurs selon votre charte graphique
✓ Ajouter numéros de page
✓ Créer table des matières automatique
✓ Ajouter notes de bas de page
✓ Insérer captures d'écran

───────────────────────────────────────────────────────────────────────
IMPRESSION
───────────────────────────────────────────────────────────────────────

Pour imprimer le guide :

Format     : A4
Orientation: Portrait
Couleur    : Oui (pour les titres et tableaux)
Pages      : ~15 pages
Recto-verso: Recommandé
